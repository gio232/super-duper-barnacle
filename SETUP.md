# 🚀 Развертывание на хостинг

## Шаг 1: Закиньте всю папку на хостинг

Загрузите всю папку проекта на хостинг через FTP или SSH.

## Шаг 2: Подключитесь по SSH

```bash
ssh username@ваш-домен.com
cd /path/to/your/site
```

## Шаг 3: Создайте .env файл

```bash
cp .env.example .env
```

Отредактируйте `.env` и установите ваши данные:
```
ADMINS=admin:ваш-пароль
PORT=8080
```

## Шаг 4: Установите зависимости

```bash
npm install
```

## Шаг 5: Запустите сервер с PM2 (рекомендуется)

```bash
# Установите PM2 глобально (если еще не установлен)
npm install -g pm2

# Запустите приложение
pm2 start backend/server.js --name "swiss-blog"

# Сохраните конфиг
pm2 save
pm2 startup

# Проверьте, что сервер запущен
pm2 list
```

## Шаг 6: Проверьте что работает

Откройте в браузере:
- **Сайт**: `https://ваш-домен.com`
- **Админ панель**: `https://ваш-домен.com/admin`
- **API**: `https://ваш-домен.com/api/posts?lang=ru`

## Полезные команды PM2

```bash
# Посмотреть логи
pm2 logs swiss-blog

# Перезагрузить приложение
pm2 restart swiss-blog

# Остановить приложение
pm2 stop swiss-blog

# Удалить приложение из PM2
pm2 delete swiss-blog
```

## Если есть проблемы

### API возвращает 404
- Убедитесь что сервер запущен: `pm2 list`
- Проверьте логи: `pm2 logs`

### Нет доступа к `/admin`
- Проверьте, что `.env` файл существует и содержит `ADMINS=admin:пароль`
- Перезагрузите сервер: `pm2 restart swiss-blog`

### Загрузка изображений не работает
- Убедитесь что есть папка `uploads` в корне проекта
- Проверьте права доступа: `chmod 755 uploads`
