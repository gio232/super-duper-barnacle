- ✅ Netlify CMS админка (`/admin`)
- ✅ Страница блога (`blog.html`)
- ✅ Скрипт автоперевода (MyMemory API - бесплатно)

---

## **Как развернуть (за 10 минут):**

### **Шаг 1: Залить на GitHub**

```bash
# Если репо еще не создано
git push -u origin main
```

Перейди на GitHub и создай публичный репозиторий: `https://github.com/твойПрофиль/каtefinalfix`

### **Шаг 2: Зарегистрироваться на Netlify**

1. Перейди на https://netlify.com
2. Зарегистрируйся через GitHub
3. Нажми "New site from Git"
4. Выбери твой репозиторий
5. **В Build settings укажи:**
   - Build command: `npm run build` (или оставь пусто)
   - Publish directory: `.` (корень проекта)

### **Шаг 3: Активировать Netlify Identity (для админки)**

1. В админ панели Netlify → Site settings → Identity → Enable Identity
2. На вкладке "Services" → Нажми "Enable Git Gateway"
3. На вкладке "Settings and usage" → Invite users → Добавь email тетки

### **Шаг 4: Готово!**

Админка будет доступна по адресу:
```
https://твойсайт.netlify.app/admin
```

Тетка заходит с email → пишет статью → публикует → статья сразу появляется на сайте на 3 языках!

---

## **Как пользователю писать статьи:**

### **В админке (CMS):**
1. Заходит в `/admin`
2. Логинится через email
3. Нажимает "New Blog Post" → "Статьи блога"
4. **Заполняет:**
   - **Язык написания:** Выбирает русский (RU)
   - **Заголовок:** Пишет на русском
   - **Описание:** Краткое описание (1-2 предложения)
   - **Содержание:** Статья в Markdown
   - **Изображение:** Загружает картинку (200x150px или больше)
   - **Опубликовано:** Галочка ☑
5. **Нажимает "Publish"**

### **Что происходит автоматически:**
- ✅ Статья сохраняется в GitHub в папке `_posts/blog/`
- ✅ Запускается автоперевод (RU → EN, DE)
- ✅ Создаются три файла: `статья.md`, `статья.en.md`, `статья.de.md`
- ✅ На сайте статья появляется на всех трех языках!

## **Если автоперевод не работает:**

Используй альтернативу через Google Sheets (более надежно):

1. Создай Google Sheet со столбцами: `RU Title | RU Content | EN Title | EN Content | DE Title | DE Content`
2. Используй бесплатное расширение: [MacroGPT](https://workspace.google.com/marketplace) для автоперевода
3. Или переводи вручную в Google Translate
