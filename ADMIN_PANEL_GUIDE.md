# 📋 Админ Панель — Полное руководство

Полностью функциональная админ панель для управления мультиязычным блогом (RU, EN, DE) с поддержкой загрузки медиафайлов и автоматического перевода.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка переменных окружения (`.env`)
```
PORT=3000
NODE_ENV=development

# Администраторы (формат: username:password|username:password)
ADMINS=admin:12345|moderator:password123

# Email для восстановления пароля (опционально)
ADMIN_EMAIL=admin@example.com
RESET_TOKEN_SECRET=your_secret_key_here_change_before_production

# Папка для загрузки файлов
UPLOAD_DIR=uploads

# Максимальный размер файла (в MB)
MAX_FILE_SIZE=10
```

### 3. Запуск сервера
```bash
npm start
```

Сервер будет доступен по адресу: **http://localhost:3000/admin-panel**

## 🔐 Аутентификация

### Вход в систему
1. Перейди на **http://localhost:3000/admin-panel**
2. Введи **имя пользователя** и **пароль**
3. Нажми **"Войти"**

Админ панель использует токены на базе:
- **Формат токена**: Base64 кодированная строка `timestamp:username:password`
- **Время жизни токена**: 24 часа

### Забыл пароль?
1. На странице логина нажми **"Забыл пароль?"**
2. Введи своё имя пользователя
3. Нажми **"Отправить инструкции"**
4. Система отправит инструкции на email (при наличии SMTP конфигурации)

### Выход из системы
Нажми кнопку **"Выход"** в правом углу админ панели.

## 📝 Создание и управление статьями

### Создание новой статьи

#### 1. Выбор языка
- Выбери язык, на котором ты пишешь статью
- Остальные языки будут переведены **автоматически**
- Доступные языки: 🇷🇺 Русский, 🇬🇧 English, 🇩🇪 Deutsch

#### 2. Заполнение основной информации
- **Заголовок статьи**: Основной заголовок (обязательно)
- **Краткое описание**: 1-2 предложения для списка статей (опционально)
- **Автор**: По умолчанию "Swiss Center Services AG" (можно изменить)

#### 3. Загрузка изображения
Есть два способа:

**Способ 1: Загрузить файл с устройства**
1. Нажми кнопку **"📤 Загрузить изображение"**
2. Выбери файл с компьютера (JPG, PNG, WebP)
3. Система автоматически оптимизирует изображение и создаст 3 версии:
   - 📐 **Original**: 1200x800 WebP (качество 80)
   - 📐 **Thumb**: 400x300 WebP (качество 75)
   - 📐 **Mobile**: 600x400 WebP (качество 75)

**Способ 2: Использовать URL**
- Введи прямую ссылку на изображение в поле **"или введи URL изображения"**

#### 4. Написание контента

##### Поддерживаемый Markdown:
```markdown
# Заголовок H1
## Заголовок H2
### Заголовок H3

**жирный текст**
*курсивный текст*

- пункт 1
- пункт 2
- пункт 3

[ссылка](https://example.com)
```

##### Быстрые кнопки форматирования:
- **Б** - Жирный текст
- **К** - Курсивный текст
- **H1** - Заголовок
- **↳** - Маркированный список
- **🔗** - Ссылка

#### 5. Публикация

**Три варианта действий:**

1. **💾 Черновик** - Сохранить как черновик (статья не видна публике)
2. **📤 Опубликовать** - Опубликовать статью (видна публике)
3. **👁️ Предпросмотр** - Предпросмотр перед публикацией
4. **🌐 Проверить переводы** - Посмотреть все переводы перед публикацией

### Проверка переводов перед публикацией

1. Заполни заголовок и содержание статьи
2. Нажми **"🌐 Проверить переводы"**
3. Откроется модальное окно со всеми тремя языками:
   - 🇷🇺 **Русский** - оригинальный текст
   - 🇬🇧 **English** - автоматический перевод
   - 🇩🇪 **Deutsch** - автоматический перевод

⚠️ **Важно**: Автоматические переводы выполняются через MyMemory API. Всегда проверяй качество перевода перед публикацией!

### Управление опубликованными статьями

В разделе **"📋 Мои статьи"** видны все созданные статьи:

- **Статус**: Показывает, опубликована ли статья или в черновиках
- **Языки**: Список языков, в которых доступна статья
- **Дата**: Дата создания статьи
- **✏️ Редактировать**: Откроется форма для редактирования (в разработке)
- **🗑️ Удалить**: Удалить статью полностью

## 📤 Загрузка файлов

### Загрузка изображений
```bash
POST /api/upload/image
Authorization: Bearer <token>
Content-Type: multipart/form-data
Body: { file: <binary> }
```

**Ответ:**
```json
{
  "success": true,
  "data": {
    "name": "unique-id-timestamp",
    "original": "/uploads/unique-id.webp",
    "thumb": "/uploads/unique-id-thumb.webp",
    "mobile": "/uploads/unique-id-mobile.webp",
    "size": 5420,
    "type": "image/png"
  }
}
```

### Загрузка других файлов
```bash
POST /api/upload/file
Authorization: Bearer <token>
```

Поддерживаемые типы: PDF, MP4, WebM, изображения

## 🌐 API Endpoints

### Аутентификация

**Логин**
```bash
POST /api/auth/login
Content-Type: application/json
Body: { username, password }
```

**Проверка токена**
```bash
POST /api/auth/verify
Authorization: Bearer <token>
```

**Сброс пароля**
```bash
POST /api/auth/forgot-password
Content-Type: application/json
Body: { username }
```

### Посты

**Получить все посты**
```bash
GET /api/posts
GET /api/posts?lang=ru
GET /api/posts?published=true
```

**Получить конкретный пост**
```bash
GET /api/posts/:slug
```

**Создать пост**
```bash
POST /api/posts
Authorization: Bearer <token>
Content-Type: application/json
Body: {
  title: string,
  description?: string,
  content: string (markdown),
  lang: "ru" | "en" | "de",
  image?: string (URL),
  author?: string,
  published?: boolean (default: false)
}
```

**Обновить пост**
```bash
PUT /api/posts/:slug
Authorization: Bearer <token>
```

**Удалить пост**
```bash
DELETE /api/posts/:slug
Authorization: Bearer <token>
```

## 🔒 Безопасность

### Перед развёртыванием в продакшене:

1. **Измени пароли администраторов** в `.env`:
   ```
   ADMINS=admin:strongpassword123|moderator:differentpassword456
   ```

2. **Установи надёжный секрет** для токенов:
   ```
   RESET_TOKEN_SECRET=generate_a_random_secure_string
   ```

3. **Используй HTTPS** в продакшене (не HTTP)

4. **Включи CORS** только для доверенных доменов:
   ```javascript
   app.use(cors({
     origin: ['https://yourdomain.com'],
     credentials: true
   }));
   ```

## 🔧 Решение проблем

### Сервер не запускается
```bash
# Проверь, что порт не занят
lsof -i :3000

# Очисти node_modules и переустанови
rm -rf node_modules
npm install
```

### Изображения не загружаются
```bash
# Проверь права доступа на папку uploads
chmod 755 uploads

# Убедись, что путь к изображению правильный
curl http://localhost:3000/uploads/filename.webp
```

### Токен истекает
- Токены действуют 24 часа
- Если токен истёк, перелогинься в админ панели
- Время жизни токена можно изменить в `backend/middleware/auth.js`

### Переводы некорректны
- MyMemory API использует простые алгоритмы перевода
- Всегда проверяй переводы перед публикацией
- Для важных текстов используй профессиональный переводчик
